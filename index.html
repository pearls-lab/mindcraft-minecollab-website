Abstract

Collaboration pushes the boundaries of what is possible. The same is true for LLM agents that work in tandem with humans or other agents. Unfortunately, the most common LLM collaboration is via disembodied chat interfaces that are far distant from complex real world embodiment. To better understand how LLM agents perform in an embodied multi-agent system, we introduce \mindcraft{} a platform that allows LLM agents to control Minecraft and MineCollab\footnote{our website here\url{}} - a benchmark we created to test the different dimensions of embodied and collaborative reasoning. Though we hypothesize that collaboration can make embodied agents be more successful, we find that communication is a bottleneck for collaboration in our tasks, dropping by over 15\% across all models when the agents are forced to communicate hidden plan information. This highlights the need for new methods beyond prompting and simple finetuning for embodied LLM reasoning and multi-agent collaboration through natural language communication.
